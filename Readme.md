## Dicionário Git do QA

| Termo            | Significado                                                             |
| :--------------- | :---------------------------------------------------------------------- |
| **Branch** | Uma cópia do projeto para desenvolver testes de forma isolada.         |
| **Commit** | Registro oficial de uma alteração feita no código (um "salvamento"). |
| **Push**   | Envio das suas alterações locais para o servidor remoto.              |
| **Pull**   | Atualização do seu computador com as últimas mudanças do servidor.  |
| **Merge**  | Fusão de uma branch de trabalho com a branch principal.                |

## Comandos Favoritos

```bash
git clone
git commit
git push
git pull
```
